⚠ Key Design Decisions in This Mockup

  • Today = Feed + Tasks. The Feed isn't gone — it's the "Journal" section inside Today. Today's tasks appear above today's journal entries. Previous days scroll below with day dividers.
  • Tab bar preserved. Notes open as tabs in the main area, exactly like current DailyDispatch. The + button creates a new note tab. Today is always the leftmost tab (like Feed was).
  • Clicking a note in the sidebar opens it as a tab (or switches to its tab if already open). Same UX as now.
  • Sidebar replaces the current topbar for primary navigation. Topbar logo moves to sidebar header. Calendar moves from left sidebar widget to the sidebar.
  • Settings, Gallery, Floating Note are footer icons in the sidebar — small, out of the way, always accessible. Search is also in the tab bar icons (Cmd+K).
  • Sections (Help, Meetings, custom filters) would move to a right sidebar panel like current DailyDispatch, or could be accessed via the Tasks tab's grouping. TBD.
  • Task checking in Today view syncs to the Kanban board (Done column). The column pill ("Inbox") shows where the task lives on the board.
  • Calendar click navigates to that day — Today view becomes that day's view with its tasks and journal entries.
