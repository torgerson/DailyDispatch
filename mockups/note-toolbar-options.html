Note Editor + Formatting Toolbar — 5 Options

Redesigning the note page: cleaner layout, formatting toolbar for discoverability, consistent with the rest of the app. Each option shows the same note content.

1 — Fixed Top Toolbar with Grouped Buttons
A full-width toolbar sits between the tab bar and the editor. Formatting buttons grouped by type: text style, headings, lists, inserts. Action buttons (Copy/Link/Delete) sit at the far right. Toolbar is always visible. Like Google Docs or Notion's fixed bar.

2 — Toolbar Inside the Editor Card
The toolbar is part of the editor card itself — sits at the top of the bordered cream area. Feels integrated, like Notion's inline toolbar. Actions (Copy/Link/Delete) move to a subtle row above the editor. The toolbar + content are one visual unit.

3 — Floating Toolbar on Text Selection
No permanent toolbar. When you select text, a small floating toolbar appears above the selection (like Medium/Notion's inline popup). Heading/list/insert actions available via the existing slash command menu. The editor is clean and distraction-free until you need formatting. Actions sit in a minimal top row.

4 — Slim Sticky Toolbar + Constrained Width
A thin toolbar bar sticks to the top as you scroll. Icons only — no labels. Max-width constraint (740px) on the editor for comfortable reading. Actions integrated into the right side of the toolbar. Compact and professional — like Bear or iA Writer's minimal chrome.
