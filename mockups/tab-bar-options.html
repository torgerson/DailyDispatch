Tab Bar — Reducing Browser Tab Confusion

The current tab bar looks too much like browser tabs. Three approaches to make app tabs visually distinct from the browser chrome above.

Option A — Breadcrumb Path Bar
Tabs become a flat breadcrumb-style path bar with subtle separators. No raised/bordered tab shapes. Active item gets an accent pill highlight. Feels like a file path or workspace switcher — visually distinct from browser tabs.

Option B — Pill / Chip Bar
Tabs are rounded pills on a tinted background strip. Active pill has a white fill + border + shadow. Very different from rectangular browser tabs — reads more like a toolbar or filter bar. The tinted background creates clear separation from browser chrome.

Option C — No Tab Bar: Open Docs in Sidebar
Eliminate the tab bar entirely. Open documents appear as a list in the sidebar under an "Open" section. The main area shows a minimal title bar with the current document name. Completely avoids tab confusion — no tabs at all. The sidebar becomes the workspace navigator.
